In python, a set is a data structure that stores an unordered collection of unique elements. The elements of a set must be immutable and hashable, and cannot be repeated. Sets are useful for performing mathematical operations on sets, such as union, intersection and difference, and for removing duplicates from a list.

## Creating a set

Let's create some sets of fruits

<Code
code="
# List of fruits
set1 = {'apple', 'banana', 'pear'}
set2 = {'pear', 'grape', 'mango'}
print(set1)
print(set2)
"
lang="py"
title="set.py"
mark="highlights"
frame='auto' />

Output:

<Code
code="
{'apple', 'banana', 'pear'}
{'pear', 'grape', 'mango'}
"
title="terminal"
mark="highlights"

## Adding elements

To add an element to a set, you can use the `.add()` method.
<Code
code="
# List of fruits
set1 = {'apple', 'banana', 'pear'}
set2 = {'pear', 'grape', 'mango'}

# Add elements to a set

set1.add('strawberry')
set2.add('guava')

print(set1)
print(set2)
"
lang="py"
title="set.py"
mark="highlights"
frame='auto' />

Output:

<Code
code="
{'apple', 'banana', 'pear', 'strawberry'}
{'pear', 'grape', 'mango', 'guava'}
"
title="terminal"
mark="highlights"
frame='terminal' />

## Deleting an element

To delete an element from a set, you can use the `.remove()` method.

## Set Unions

You can join the elements of a set using the `.union()` function

<Code
code="
# List of fruits
set1 = {'apple', 'banana', 'pear'}
set2 = {'pear', 'grape', 'mango'}

# Adding elements to a set

unions = set1.union(set2)

print(unions)

"
lang="py"
title="set.py"
mark="highlights"
frame='auto' />

Output:

<Code
code="
{'apple', 'banana', 'pear', 'grape', 'mango'}
"

## Set Intersections

You can extract the elements that are repeated between sets using the `.intersection()` function

<Code
code="

set2 = {'pear', 'grape', 'mango'}

# Adding elements to a set

intersection = set1.intersection(set2)

print(intersection)

"

## Set differences

You can obtain the elements that are in one set but not in another using the `.difference()` function.
This is useful for identifying which values are exclusive to a set.

<Code
code="
# List of fruits
set1 = {'apple', 'banana', 'pear'}
set2 = {'pear', 'grape', 'mango'}

# Adding elements to a set

difference = set1.difference(set2)

print(difference)

"
lang="py"
title="set.py"
mark="highlights"
frame='auto' />

Output:

<Code
code="
{'apple', 'banana'}
"
