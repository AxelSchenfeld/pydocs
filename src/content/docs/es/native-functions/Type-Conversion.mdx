En Python, los tipos de datos se utilizan para clasificar un tipo específico de datos, determinar los valores que puede asignar al tipo y las operaciones que puede realizar en el mismo. Cuando realice tareas de programación, a veces, deberá aplicar conversiones de valores entre tipos para manipular los valores de forma diferente. Por ejemplo, es posible que debamos concatenar valores numéricos con cadenas o representar posiciones decimales en números que se iniciaron como valores enteros.

Los tipos son:

- `int()`
- `float()`
- `str()`
- `bool()`
- `list()`
- `tuple()`
- `set()`
- `dict()`

### `int()`

Convierte un valor numérico o una cadena que representa un número entero al tipo de dato int. Si el valor es un número decimal, elimina los decimales. Si no es numérico, lanza un error.

<Code 
code="
print(int(43.54)) # Convierte el decimal a entero, quitando los decimales
"

<Code 
code="
43 
"

mark="highlights"
frame='terminal' />

<Aside type="note">
 No es posible convertir directamente una cadena que representa un número decimal (float) a un entero usando int(). int("3.14") lanza ValueError.
</Aside>

### `float()`

Convierte un número entero o una cadena que representa un número en un número decimal (float), Si no es numérico, lanza un error. Útil cuando se necesita trabajar con valores fraccionarios.

print(float(10))

### `bool()`
Convierte un valor a True o False según si es "vacío" o no. Valores como 0, "" (cadena vacía), [] (lista vacía), None y False se consideran False. Todo lo demás es True.
<Code 
code="

print(bool(0))
print(bool('Hola'))
"

### `set()`

set() convierte un objeto iterable en un conjunto, eliminando duplicados y sin mantener un orden específico (es mutable). 

<Code 
code="
print(set([1, 2, 2, 3]))
print(set('hello') )
"
lang="py" 
title="Set.py" 
mark="highlights"
frame='auto' />
Salida: 
<Code 
code="
{1, 2, 3}
{'h', 'e', 'l', 'o'}
"
title="Set.py" 
mark="highlights"
frame='terminal' />

<Aside type="note">
Para que set() funcione, los elementos del iterable deben ser hashables, lo que significa que deben tener un valor hash único.
Ejemplos de objetos no hasheables:
- Listas
- Conjuntos
- Diccionarios
</Aside>

### `dict()`

Convierte un objeto iterable en un diccionario.

<Code 
code="
print(dict([('a',1), ('b',2)]))  
print(dict(a=1, b=2))            
"
lang="py" 
title="Dict.py" 
mark="highlights"
frame='auto' />
Salida: 
<Code 
code="
{'a': 1, 'b': 2}
{'a': 1, 'b': 2}
"
title="Dict.py" 
mark="highlights"
frame='terminal' />

<Aside type='note'>
  dict() solo funcionará en los siguientes casos:
  - Objetos iterables que contengan pares clave-valor.
  - Objetos iterables que contengan tuplas, listas o conjuntos de dos elementos.
  - Objetos iterables que contengan diccionarios.
  - Las claves deben ser unicas y hashables.
</Aside>
