# Funciones básicas de Python

Python ofrece una variedad de funciones integradas que facilitan las tareas de programación cotidianas. Estas funciones están disponibles sin necesidad de importar módulos adicionales.

## Uso de las funciones

Las funciones integradas son parte del núcleo de Python y pueden ser llamadas directamente sin necesidad de importar módulos o bibliotecas adicionales.

```python
# Ejemplo de uso de varias funciones básicas
nombre = input("¿Cómo te llamas? ")
print(f"Hola, {nombre}!")

mi_lista = list(range(1, 6))
print(f"Lista generada: {mi_lista}")
print(f"Longitud de la lista: {len(mi_lista)}")
```

<div class="tip">
  <strong>Consejo:</strong> Para obtener ayuda sobre cualquier función integrada, usa la función <code>help()</code>.
  Por ejemplo: <code>help(print)</code> mostrará la documentación de la función print.
</div>
