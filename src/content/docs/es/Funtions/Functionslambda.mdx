En Python, una función lambda es una función anónima (sin nombre) definida en una sola línea usando la palabra clave <code>lambda</code>. Son útiles para operaciones cortas, como transformar datos o filtrar elementos, especialmente al usar funciones de orden superior como <code>map()</code>, <code>filter()</code> y <code>sorted()</code>. 

## Características de las funciones lambda:

- **Anónimas**: No tienen nombre definido. 

- **Sintaxis simple**: Se definen con <code>lambda</code> seguido de argumentos, <code>:</code> y una expresión. 

- **Una sola expresión**: Solo pueden contener una única expresión para el resultado. 

- **Uso con funciones de orden superior:** Son comunes con <code>map()</code>, <code>filter()</code> y <code>sorted()</code>.
