Cuando escribes un programa, no todo debe quedar dentro del código. A veces necesitas mostrar información al usuario (salida), y otras veces pedirle datos (entrada).En Python, esto se hace principalmente con dos funciones muy sencillas:

- `print()` para mostrar mensajes o resultados.

- `input()` para recibir datos del usuario.

Con estas dos funciones puedes empezar a construir programas interactivos.

## Print()

La función `print()` en Python se usa para mostrar información en la consola. Permite imprimir texto, números, variables, y más, utilizando una sintaxis sencilla.
